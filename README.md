# Portfólio Profissional: Lucas Froner

Este projeto consiste no desenvolvimento de um portfólio digital personalizado para **Lucas Froner**, administrador e criador do projeto *Pelas Ruas de Porto Alegre*. O objetivo principal foi criar uma plataforma moderna e eficiente para a divulgação de sua trajetória profissional e projetos autorais.

## Tecnologias Utilizadas

Para garantir uma interface leve, responsiva e de alto desempenho, foram utilizadas as tecnologias fundamentais do desenvolvimento web:

* **HTML5**: Estruturação semântica e acessível do conteúdo.
* **CSS3**: Estilização avançada, garantindo um layout responsivo e visual profissional.
* **JavaScript**: Implementação de interatividade e melhoria na experiência do usuário.
